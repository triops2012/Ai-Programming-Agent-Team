# Registro de Decisiones Técnicas (ADR)

## ADR-01: Estructuración del Archivo README para GitHub
- **Fecha:** 2026-06-26
- **Estado:** PROPUESTO
- **Contexto:** El usuario Raúl Navas Montero necesita un archivo `readme.md` profesional para la publicación del Proyecto Atenea en GitHub. El archivo original de AI Studio solo contiene un banner mínimo.
- **Decisión:**
  - Crear un archivo `/github/README.md` (o `/github/readme.md`) de alta calidad técnica y visual.
  - El diseño utilizará un estilo estructurado, iconografía integrada, y formato markdown limpio.
  - El contenido detallará los pilares técnicos (Sistemas de Agentes, Memoria RAG en IndexedDB, Modo Autónomo, Gamificación) basándose en la documentación de ingeniería de la aplicación.
  - Se agregará la sección de autoría con enlaces directos a su LinkedIn y descripción de su rol como creador y diseñador principal.
- **Consecuencias:**
  - El repositorio de GitHub tendrá una carta de presentación estelar, aumentando el impacto profesional de Raúl Navas Montero.
  - La ubicación en `/github/README.md` permitirá mantener el código del proyecto limpio, mientras se tiene la documentación de publicación en su propia carpeta especializada.
